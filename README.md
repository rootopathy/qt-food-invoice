# Q&T Food Invoice Maker

Static browser app for creating Q&T food invoices from manual entry or Excel upload.

## Features

- Invoice preview matching the provided sample format
- One invoice per Excel row
- Previous/Next navigation for uploaded Excel batches
- IGST or CGST + SGST calculation
- Editable GST amount and round-off
- Invoice history in browser storage
- Individual PDF download and Download All

## Run Locally

Open `index.html` in a browser, or serve this folder with any static file server.
